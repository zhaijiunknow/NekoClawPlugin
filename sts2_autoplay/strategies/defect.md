# 故障机器人（鸡煲）策略文档

## 角色定位
- 鸡煲就是故障机器人。
- 这是一个极度偏防守反击、强调低战损比的角色。
- 核心弱点是启动慢、费用高、纯靠单卡数值硬撑，卡组内部天然联动不如其他角色顺滑。
- 因为缺少稳定的烧牌、猎宝式过牌弃牌体系，鸡煲必须更重视起防、过牌和整体卡组运转。

## 总体思路
1. 优先保证起防，降低掉血。
2. 输出主要依靠能量球体系，尤其是电球。
3. 不盯着单卡强度抓牌，而是根据当前三选一动态补足卡组短板。
4. 越早确定流派越好，但各流派之间可以混搭，不需要一条路走到黑。

## 基础战斗原则
- 前期鸡煲不急着靠打击输出，电球的效率通常更高。
- 两个电球在场就能稳定提供接近 1 费打击的输出质量。
- 配合双重释放等牌，电球爆发明显优于普通打击。
- 因此前期可以更积极地削减打击，甚至在极端情况下连部分防御也能删，但前提是已经有稳定起防与球体系支撑。

## 抓牌核心优先级

### 必抓/高优先级
这些牌通常可以直接提高鸡煲的起防、运转或核心输出：
- 冷头：起防同时过牌，优质启动牌。
- 快速检索：补运转、找关键牌。
- 全息影像：回收关键组件，缓解鸡煲卡手。
- 暴风雨：去掉消耗后极强，0 费产球基本不亏。
- 冰川：高质量起防，提供大量防御。
- 充电：没有冰川等强起防卡时，抓一张通常不亏。
- 高速脱离：状态鸡优质防卡，副作用可接受，还能联动部分状态收益牌。
- 白噪声：中后位必抓卡；前两张都一般时，抓白噪声通常安全。

### 推荐抓取原则
- 优先补过牌、起防、集中、球体系。
- 如果当前卡组启动差，就优先抓冷头、冰川、充电、高速脱离这类稳定牌。
- 如果当前已经有足够防御，再考虑抓流派核心继续拔高上限。

## 不推荐盲抓
- 不要死盯某一张“神卡”，而是看当前候选三张里谁最能补足卡组。
- 没有压缩时，不推荐抓硬撑、超频等会增加状态负担的牌，容易卡手。
- 对于纯数值高但进一步拖慢节奏、又不改善运转的高费牌，要谨慎。

## 流派与核心牌

### 雷暴鸡 / 能力球体系
核心思路：利用能力牌和电球持续滚雪球。

优先核心：
- 引雷针：免费多回合产球并提供不错护甲，能敲优先敲。
- 电流相生：重要核心。
- 子程序 + 雷暴：经典核心组合。
- 创造性AI：补能力来源，让体系滚起来。
- 超临界态 + 双倍：适合搭配内核加速、全息影像、暴风雨完成爆发或叠层。
- 内核加速：帮助启动与爆发链条成型。

抓牌方向：
- 多抓能力牌
- 多抓产球牌
- 多抓能回收关键牌、提高运转的牌

### 状态鸡
核心思路：接受一定状态负担，换取高质量防御与收益联动。

优先核心：
- 高速脱离：最简单、负面最低的状态防卡。
- 火箭拳：过牌与输出兼具，和高速脱离联动很好。
- 污秽攻击：如果黏液副作用可接受，则可以作为补充输出与体系件。
- 压缩：状态鸡关键缓冲器，没有它时不要乱抓状态负担牌。

抓牌方向：
- 先确认自己有没有压缩或足够的状态消化能力。
- 有压缩时可适当接受状态牌；没有压缩时要保守。

### 0 费物理鸡
核心思路：围绕低费物理输出快速成型。

优先核心：
- 关键 0 费核心牌是上限来源，拿到后可以明显偏向这一流派。
- 羽化、万众一心可根据当前卡组形态决定是否补入。

抓牌方向：
- 如果候选里出现明确核心，可考虑转型。
- 但若当前球体系已经成型，不必硬转。

## 动态选牌规则
自动选牌时，不应机械按固定榜单抓牌，而要看当前候选项是否满足以下原则：

1. **起防优先**
   - 如果当前缺少稳定防御，优先冷头、冰川、充电、高速脱离。

2. **运转优先**
   - 如果当前卡组容易卡手，优先快速检索、全息影像、白噪声等改善运转的牌。

3. **球体系优先**
   - 如果候选牌能补强电球/能力球体系，例如暴风雨、雷暴、引雷针、电流相生、子程序，则优先度很高。

4. **状态牌要看前提**
   - 如果当前牌库没有压缩，就降低硬撑、超频等状态负担牌优先级。

5. **同组三选一卡里择优补短板**
   - 三张都不错时，优先选最能补当前短板的那张，而不是永远抓同一类牌。

6. **没有明显优质牌时保守处理**
   - 如果三张都一般，优先低风险、不卡手、能补基础功能的牌。

## 运营建议

### 第一层
- 推荐至少打一场精英，除非开局特殊奖励让避战收益明显更高。
- 普通怪也要适当多打，尽量在第一层尽早拿到核心牌并确定方向。
- 问号房走两三个通常够了，不要过度贪问号导致成型变慢。

### 第二层
- 是卡组巩固期。
- 围绕已确定流派决定多打怪拿牌，还是走问号删卡、拿关键事件收益。
- 如果发育顺利，可以多打精英；完整体鸡煲对第二层精英压力相对可控。

### 第三层
- 更推荐多走问号、少打普通怪。
- 某些针对球体系的小怪会让鸡煲掉很多血，收益不一定划算。
- 重点放在保战损、准备 Boss 战。

## 黑球与冰球态度
- 当前版本更推荐通过一回合多次主动破球来灌伤。
- 因此黑球与冰球在多数构筑中的出场率不如电球体系。
- 除非后续补强明显，否则现阶段优先围绕电球与能力体系构筑。

## 给自动策略的落地要点
- 鸡煲 = 故障机器人。
- 角色选择优先故障机器人，不能选时再选铁血战士。
- 选牌时以“防御启动、运转、电球体系、流派核心、状态负担控制”为主要评分维度。
- 不机械追单卡，而是根据当前候选动态选择最适合的一张。
