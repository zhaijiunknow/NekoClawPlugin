# Suno.cn 音乐生成插件

N.E.K.O 的 Suno.cn AI 音乐生成插件，通过对话即可生成原创音乐。

## 功能概览

- **生成音乐**：根据描述或歌词生成 AI 音乐
- **查询任务**：查看音乐生成进度和结果
- **获取歌词**：提取已生成歌曲的歌词
- **续写音乐**：延长现有歌曲
- **AI 写词**：根据灵感自动生成歌词
- **历史列表**：查看已生成的音乐记录
- **账户信息**：查询积分和会员状态
- **播放音频/打开链接**：在浏览器中播放生成的音乐或打开任意网页链接

## 配置

编辑 `plugin.toml`：

```toml
[suno]
api_key = "sk-your-api-key-here"
base_url = "https://mcp.suno.cn"
timeout_seconds = 60
```

或通过对话设置：
```
帮我设置 Suno API Key 为 sk-xxxxx
```

## 使用方式


### 第一步：获取 API Key

1. 访问 https://www.suno.cn/home/#/mcp
2. 登录后点击「AI 助手接入」→「创建新密钥」
3. 复制 sk- 开头的密钥
4. ⚠️ 立刻保存，关闭后无法再查看

---

### 第二步：配置密钥

在 OpenClaw 的「Skills」中找到 suno-cn-music，在 SUNO_CN_API_KEY 填入密钥。

或手动编辑 ~/.openclaw/config.json：
{"env":{"SUNO_CN_API_KEY":"sk-你的密钥"}}

---

### 第三步：开始创作

对话示例：
- 「帮我生成一首关于夏天的流行歌曲」
- 「查一下刚才的歌好了吗」
- 「显示我最近的音乐列表」
- 「获取这首歌的歌词」
- 「播放刚才生成的歌」
- 「打开链接 https://example.com」


## API 调用规范

### 1. 生成音乐

**基础用法（推荐）**

直接描述你想要的音乐风格和主题：

```
生成一首关于春天的轻快歌曲
```

```
帮我做一首悲伤的钢琴曲
```

**参数说明**

| 参数 | 类型 | 必填 | 说明 | 推荐值 |
|------|------|------|------|--------|
| `prompt` | 字符串 | ✅ | 音乐描述或完整歌词 | 简短清晰的中文描述 |
| `title` | 字符串 | ❌ | 歌曲标题 | 留空让 AI 自动生成 |
| `tags` | 字符串 | ❌ | 风格标签 | 留空或简单英文词 |
| `custom_mode` | 布尔 | ❌ | 自定义歌词模式 | `false`（默认） |
| `instrumental` | 布尔 | ❌ | 纯音乐模式 | `false`（默认） |
| `mv` | 字符串 | ❌ | 模型版本 | `v5`（默认，最新最稳定） |

**工作流程**

1. 提交生成请求后立即返回任务编号
2. 后台自动轮询生成进度（约 30-60 秒）
3. 完成后主动通知并提供播放链接

**注意事项**

- ✅ **推荐**：只传 `prompt`，让 API 自动处理其他参数
- ✅ **推荐**：使用默认模型 `v5`
- ✅ **推荐**：不要自动补充 `title`、`tags`，除非用户明确要求
- ⚠️ **谨慎**：`custom_mode=true` 仅在用户提供完整歌词时使用
- ⚠️ **谨慎**：复杂的 `tags` 组合可能导致解析失败
- ❌ **避免**：同时传入过多参数

### 2. 查询任务状态

```
查询任务 2037286812988215296 的状态
```

返回信息包括：
- 任务状态（queued / processing / success / failed）
- 歌曲标题
- 播放链接（成功时）
- 失败原因（失败时）

### 3. 获取歌词

```
获取 2037286812988215296 的歌词
```

返回完整歌词文本，不做任何修改或翻译。

### 4. 续写音乐

```
续写任务 2037286812988215296，从第 20 秒开始
```

参数：
- `serial_no`：原任务编号
- `continue_at`：从第几秒开始续写（默认 0）
- `mv`：模型版本（默认 `v5`）

### 5. AI 生成歌词

```
帮我写一首关于友情的歌词
```

```
生成一首摇滚风格的歌词，主题是自由
```

参数：
- `inspiration`：灵感描述
- `title`：歌曲标题（可选）
- `style`：风格（可选）

生成后会询问是否直接用这个歌词生成音乐。

### 6. 查看历史

```
查看我的音乐列表
```

```
查看第 2 页的音乐记录
```

### 7. 账户信息

```
查询我的 Suno 账户信息
```

返回：
- 昵称
- 剩余积分
- 会员状态

### 8. 播放音频/打开链接

**播放最近生成的音乐**

```
播放刚才生成的歌
```

插件会自动打开最近一次生成的音乐链接。

**播放指定音乐**

```
播放任务 2037286812988215296 的音乐
```

**打开任意网页链接**

```
打开链接 https://example.com
```

支持打开任意 `http/https` 链接，不限于音频。


## 常见问题

### Q: 如何生成纯音乐（无人声）？

```
生成一首纯音乐的钢琴曲，不要人声
```

或明确指定：
```
instrumental=true
```

### Q: 如何使用自己写的歌词？

```
用这个歌词生成音乐：
[你的完整歌词]
```

或明确指定：
```
custom_mode=true, prompt="[完整歌词]"
```

### Q: 积分不足怎么办？

查询账户信息确认剩余积分：
```
查询 Suno 账户
```

如需充值，访问 [Suno.cn 官网](https://suno.cn)。

## 技术细节

### API 端点

| 功能 | 方法 | 端点 |
|------|------|------|
| 账户信息 | GET | `/mcp/api/user` |
| 生成音乐 | POST | `/mcp/api/generate` |
| 查询任务 | GET | `/mcp/api/task/{serial_no}` |
| 音乐列表 | GET | `/mcp/api/music` |
| 获取歌词 | GET | `/mcp/api/lyrics/{serial_no}` |
| 续写音乐 | POST | `/mcp/api/extend` |
| 生成歌词 | POST | `/mcp/api/gen-lyrics` |

### 任务状态

- `queued`：排队中
- `processing`：生成中
- `success`：成功
- `failed`：失败

### 后台轮询机制

- 提交后立即返回，不阻塞对话
- 后台每 10 秒轮询一次，最多 6 次（约 60 秒）
- 完成后通过主动通知推送结果
- 失败时显示具体错误原因

## 许可与限制

- 本插件仅封装 Suno.cn API，不修改任何返回内容
- 歌词和标题原样返回，不做翻译或总结
- 遵循 Suno.cn 服务条款和使用限制
- 生成的音乐版权归 Suno.cn 及用户所有
